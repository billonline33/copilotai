# To-Do List App

This project is a Next.js to-do list application scaffolded with TypeScript, ESLint, and an empty starter. Follow the technical checklist in `/doc/03_TechnicalChecklist.md` for implementation steps.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `/src` - Main source code
- `/doc/03_TechnicalChecklist.md` - Implementation checklist
